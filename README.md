# TextUtils
A tool for analyzing text data in Django backend

It is a simple django project or website in which we can Analyze text.

<h1>What Can We do from Textutils ?</h1>
=> Remove Punctuations<br>
=> UPPERCASE<br>
=> New Line Remove<br>
=> Extra Spaces Remover<br>
=> Numbers Remover
